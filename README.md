# Mathify Notes 🧪🧮

**Mathify Notes** is a smart Obsidian plugin designed for IGCSE Science & Mathematics students to make digital note-taking as fast as writing on paper. It automatically converts simple keyboard shorthands into beautiful formatted math symbols, arrows, and upright IUPAC chemical formulas in real-time as you type!

## 🚀 Features & Shortcuts

Just type any of the following shorthands and press **Space** to instantly format your text:

| Subject | What you type | What Mathify Notes outputs |
| :--- | :--- | :--- |
| **🧪 Chemistry** | `H2O` | $\text{H}_2\text{O}$ |
| | `Ca(OH)2` | $\text{Ca(OH)}_2$ |
| | `SO4^2-` | $\text{SO}_4^{2-}$ |
| | `->` or `-->` | $\rightarrow$ / $\longrightarrow$ |
| | `<=>` | $\rightleftharpoons$ |
| **🧮 General Math** | `1/2` | $\frac{1}{2}$ |
| | `sqrt(x+y)` | $\sqrt{x+y}$ |
| | `+-` or `-+` | $\pm$ / $\mp$ |
| | `pi` / `approx` | $\pi$ / $\approx$ |
| **🔭 Physics** | `v^2` | $v^2$ |
| | `v_i` | $v_i$ |
| | `ohm` / `micro` | $\Omega$ / $\mu$ |
| | `delta` / `theta` | $\Delta$ / $\theta$ |
| **💻 CS & Logic** | `<-` | $\leftarrow$ |
| | `!=` | $\neq$ |
| | `<=` or `=<` | $\leq$ |
| | `>=` | $\geq$ |

## ⚙️ Installation
1. Search for **Mathify Notes** in the Obsidian Community Plugins directory.
2. Click **Install**, then **Enable**.
3. Start taking notes! Shorthand will convert automatically in editing views whenever you hit the spacebar.
