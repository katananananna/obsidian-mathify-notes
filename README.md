# Obsidian Mathify Notes

A plugin for Obsidian that instantly auto-formats math and chemical formulas as you type.

## Features
- Auto-formats chemical notations like `SO42+` to $\text{SO}_{4}^{2+}$ and `H2O` to $\text{H}_2\text{O}$ instantly.
- Handles complex coordination compounds, polyatomic ions, and hydration states.

## Installation
1. Go to **Settings** > **Community plugins** in Obsidian.
2. Turn off **Safe mode**.
3. Click **Browse** and search for **Mathify Notes** (once approved).
4. Click **Install**, then **Enable**.

## Usage
Simply type standard text chemical symbols or math shorthand in any note, and the plugin will automatically typeset them into beautiful formatting.
